## Config structure
[files with `*` are files with keybindings set-up]
```
nvim
├── README.md
├── init.lua
├── lazy-lock.json
└── lua
    ├── configs
    │   ├── basic.lua
    │   ├── * keybindings.lua

    └── plugins
```
